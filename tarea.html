¿QUE ES UN PROTOCOLO HTTP?

Hipertext Transfer Protocol( Protocolo de Transferencia de Hipertextos),Nos permite intercambiar información entre un servidor web y un cliente, lo que hace este servidor es esperar peticiones y si la peticion viene con el protocolo correcto le devolvera la información solicitada.

¿QUE ES UN QUERY STRING?

UNIFORM RESOURCE LOCATOR (Cadena de Consulta),es una parte de una URL que se utiliza para enviar información adicional al servidor web, como parámetros o datos de búsqueda, El query string es una serie de pares clave-valor separados por el símbolo "&".

¿QUE ES UN VERBO HTTP?

Se refiere a los diferentes métodos de solicitud HTTP que pueden ser utilizados para interactuar con recursos en un servidor web. Los verbos mas comunes de http son: GET, POST, PUT, DELETE.

¿QUE ES UN DNS?

DOMAIN NAME SYSTEM (Sistema de Nombre de Dominio), Se utiliza para resolver las peticiones de asignación de nombre, es decir que cuando un usuario quiere acceder a una pagina web se necesita una dirección IP del servidor donde se encuntra dicha pagina. EJEMPLO:www.facebook.com IP:70.25.354.12

¿QUE ES UN CDN?

CONTENT DELIVERY NETWORK (Red de Distribución de Contenido), es una red de servidores distribuidos en diferentes ubicaciones geográficas que se utilizan para entregar contenido de internet, como imágenes, videos, archivos, aplicaciones y otros tipos de datos, de manera más rápida y eficiente a los usuarios finales.
